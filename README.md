# phpdb_backup
PHP MySQL Backup script with Codeignitter
How to use see index.php file

<br />
<b> Sample Backup code </b>
<code>

  $backup = require("backup.php");
  
  $backup->backup_tables();

</code> 

<b >Download code </b>
<code>

  $backup = require("backup.php"); 
  
  $backup->download();
</code>
